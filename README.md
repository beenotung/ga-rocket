## 遺傳算法的背景

又名「基因演算法」(Genetic Algorithm)

- 是一種「進化演算法」 (Evolutionary Algorithm)
- 在 1960 年由 University of Michigan 的 John Holland 帶領的研究團隊制定
- 適用於解決確定性算法過於耗時的問題
    - 如「旅行商問題」(Travelling salesman problem)
    - 和「背包問題」(knapsack problem)
